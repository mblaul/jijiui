jijiui

A clean, simple, and design kit for building niiice websites.

Getting Started

To get started with jijiui, simply include the CSS and JavaScript files in your project. You can customize the styles by overriding the CSS variables or by adding your own styles on top of the base styles provided by jijiui.

Components

jijiui includes a variety of pre-designed components such as buttons, cards, forms, and more. These components are built using the CSS variables defined in the core styles, making it easy to customize their appearance to fit your design needs.
